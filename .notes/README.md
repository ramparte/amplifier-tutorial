## Purpose

- **Curated explanations** - Expert summaries that clarify concepts
- **User feedback** - Things users found confusing or missing

## How to Use

When rebuilding the tutorial:
1. Review notes in this directory for new content to incorporate
2. Notes with `[INCORPORATED]` prefix have been added to the tutorial
3. Notes with `[PENDING]` need to be worked into content
